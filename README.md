# Fruit Ninja VR

这是一个面向 Meta Quest 真机运行的 VR 切水果小游戏。项目目标是在 Unity 中实现类似 Fruit Ninja 的核心体验：玩家使用手柄控制刀具，切开飞来的水果并避开炸弹，同时在 VR 场景中显示分数、倒计时和游戏反馈。

## 游戏模式

- Classic：经典模式，游戏时长 120 秒，漏掉水果会累计失误，失误达到 3 次后游戏结束。该模式适合展示最基础的切水果规则。
- Arcade：街机模式，游戏时长 60 秒，水果生成更快，炸弹出现概率更高，节奏更紧张，目标是在限定时间内尽量获得高分。
- Zen：禅模式，游戏时长 90 秒，没有炸弹，漏掉水果也不会惩罚，主要用于轻松体验切水果操作和 VR 手感。
- Battle：挑战模式，游戏时长 45 秒，水果生成最快、抛射力度更高，适合作为高难度计分挑战。

## Unity 真机启动

1. 使用 Unity 2021.3 打开项目根目录。
2. 用 USB 连接 Quest，并在头显中允许 USB 调试。
3. 在 Unity 菜单中执行 `Fruit Ninja > Configure Quest 3 Build`。
4. 打开 `File > Build Settings`，确认平台为 `Android`，设备选择已连接的 Quest。
5. 点击 `Build And Run` 启动到真机。
